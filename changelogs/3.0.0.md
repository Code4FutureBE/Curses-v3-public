## Beta Release
- Der Beta-Release ist nun live! Testet das Tool, macht eure Einstellungen und viel Spaß beim nutzen von Curses F-Tool v3!
- Wer das Projekt finanziell unterstützen möchte, schreibt mir eine DM in discord!
## Was macht die v3 besser als die v2?
- Neben dem großen visual Update auf der v3, gibt es etliche Updates die im Hintergrund optimiert worden sind, darunter:
- Schnelle und Reiblungslose Abwicklung von Hotkeys und Erkennung. Keine Probleme / Konflikte zwischen dem Senden der Tasten.
- Das Nachhängen von Tasteneingaben wurde gefixxt.
- DX9 + DX11 werden ohne Probleme unterschieden und erkannt. Egal auf welcher .exe ihr spielt, oder ein Profil erstellt.
- Das Einstellen der Hotkeys die unterdrückt werden sollen, kann nun pro Makro eingestellt werden.
- Das Overlay hat keine Bugs mehr - zuvor hat es bei einigen Nutzern, nach einem Follow-Befehl (allgemein nach einem Singlepress Makro), Loop Makros nicht mehr korrekt angezeigt.
- Alle gängigen Tasten einer 100% Tastatur sind nun supported. Tastenbelegungen für einen Hotkey supporten bis zu 2 Sondertasten + 1 normale Taste gleichzeitig. Bsp: Strg+Shift+Z
- Ein Makro -> Ihr entscheidet welche Clients diese ausführen sollen. In dieser Version werden keine Makros pro Charakter erstellt, ihr erstellt Makros und teilt dieses den Clients dann zu.
- Und zusätzlich, als kleines Dankeschön für die ganzen User, gibt es eine Vielzahl an neuen Themes! Ich habe euch einige Themes erstellt. Alle kostenlos, have fun!
